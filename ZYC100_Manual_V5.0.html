2. 产品组成与适配关系

ZYC100 系列采用控制器与射频模块分离的产品结构。用户可根据实际测试场景选择控制器、衰减模块、开关模块和上位机软件。

产品定位说明
ZYC100 控制器 核心控制器 提供 OLED 本机界面、USB CDC、USART1 AT 指令和模块控制能力。控制器本体不含电池。
ZYE660 单模块版 PE43711 衰减模块 提供 9kHz-6GHz 数字程控衰减通道，适合二次集成、备件和与 ZYC100 组合使用。
ZYE660 离线版（升级版） 便携衰减模块 内置电池，适合脱机便携测试，也可与 ZYC100 组成程控衰减系统。
ZSW100 扩展设备 属于 1+N 生态中的后续扩展设备，具体接口和射频指标以后续资料为准。
ZYA1000 桌面上位机 用于 ZYC100 设备发现、衰减控制、多设备管理、补偿、日志和自动化测试。

2.1 积木式连接

ZYC100 与 ZYE660 不再按单一固定产品理解，而是按积木式连接使用：控制器负责显示、通信和控制逻辑，射频模块负责具体射频通道。后续新增模块时，可继续接入 ZYC100 生态。

3. 安全与使用注意事项

重要安全提示：请在使用设备前仔细阅读以下注意事项。
  • 连接射频链路前，请确认信号源功率、频率范围和接头类型符合模块规格。
  • ZYE660 的 SMA 输入/输出应避免机械应力，安装和拆卸时建议使用合适扭矩。
  • ZYC100 控制器本体不含电池，不应按独立电池设备充电或拆解。
  • ZYE660 离线版（升级版）内置电池，长时间不用时建议定期补电并避免高温存放。
  • 请勿在通电状态下强行拔插衰减器模块或开关线缆，避免浪涌击穿损坏射频芯片！
  • 升级固件或运行自动化测试时，请保持连接稳定，不要随意断开 USB、串口或模块连接线。
  • 产线或批量测试前，请先在单台设备上验证衰减范围、RF 通道和自动化脚本。

4. 硬件装配指南

4.1 电源模式设定（安装前必读）

控制器背面配备电源模式切换开关 (MODE SET)，分两种工作模式：

无电池模式ZYC100 控制器本体按无电池设备理解，由 USB 或外部组合方案供电。
有电池模式仅用于接入支持电池的模块或版本，例如 ZYE660 离线版（升级版）。
先设模式再装配安装模块前确认 MODE SET、VCC SET 和模块拨码开关位置。
模式开关位置说明
无电池模式 靠近 USB 接口一侧 系统由 USB 直接供电，屏幕不显示电量，以 ZXYH 标识替代。
有电池模式 远离 USB 接口一侧 支持内置电池循环充电，可便携使用。无电池版本需直接拔除 Type-C 线缆实现断电关机。
注意：有电池模式仅适用于使用 ZYE660 离线版（升级版）等内置电池模块的场景。ZYC100 控制器本体不含电池。

4.2 衰减器模块拼接步骤（仅限衰减器用户）

（成套购买已拼装好请忽略该章节）

  1. 模式设置：安装前需将衰减器上的 PS 拨码开关设为 1，开启串行通信模式。
  2. 电压设置：VCC 为电源，电压可通过控制器背面的 VCC SET 开关设置为 VBUS 或 3.3V（VBUS 为 USB 供电电压，通常为 5.0V-5.3V，请根据实际需求选择）。
  3. 对准接口：对齐控制器与衰减器模块的排针接口，同时注意开机位置的键帽。
  4. 缓慢按压：沿垂直方向缓慢拼接，确保排针完全插入且无歪斜。
  5. 异常处理：电池版本若无法通过开关正常关机，多为开机键帽位置偏移，需轻轻分开模块，重新对齐键帽后再次拼接。

4.3 射频开关连接步骤（仅限射频开关用户）

注意：射频开关模块不支持与控制器进行物理上下拼接！
  1. 物理接线：请使用配套的 4Pin 专用线缆，一端连接控制器左上角的 PH2.0 4Pin UART 接口，另一端连接射频开关模块。线缆定义从上到下分别为：Ctrl、EN、VCC、GND。
  2. 电压设置：VCC 为输出电源，电压可通过控制器背面的 VCC SET 开关设置为 VBUS 或 3.3V（射频开关最低需要 5V 供电，请设为 VBUS 模式）。
  3. 模式切换：接线完成后，请进入控制器设置菜单，将工作模式（Mode）修改为 SWITCH。

5. 面板操作指南

5.1 设置菜单（通用）

通过垂直长按拨轮中心一秒钟可进入设置菜单，单按中心切换菜单项，左右拨动修改参数。修改完成后，再次长按拨轮中心一秒钟保存并退出。

菜单项说明
Step（默认步进）设置衰减模式下上电默认的衰减步进。
Def Att（默认衰减值）设置衰减模式下上电默认的衰减值。
Sleep（控制器休眠）设置电池供电时的自动休眠时间。到达时间屏幕关闭，低功耗运行。按任意键唤醒。（注：低电量、充电状态下休眠不生效；设为 0 为常亮）
Charge（禁用充电）设为 OFF 禁用充电功能，此时插入充电器也不会对电池充电，电源纯净无杂散，不干扰高精度测量。
Mode（控制模式）设为 ATT：驱动数字衰减器。设为 SWITCH：驱动 SPDT 射频开关。
Mid Key（中键功能）仅在 SWITCH 模式下生效。设为 EN/DIS：主界面按下拨轮中心用于切换开关的使能/禁用。设为 CH 1/2：主界面按下拨轮中心用于快速切换 RF1/RF2 通道。
About（关于）向右拨动进入设备详情页，查看硬件版本、UID、基本 AT 指令提示等。

5.2 衰减器控制操作（仅 ATT 模式）

ATT 模式用于控制数字衰减模块。日常只需要理解三个动作：调衰减值、改步进、进入菜单保存设置。

OLED 显示示意（ATT 模式）
-10.25 dB
Step: 0.25 | REL
  • 快速调节：向右拨动拨轮增加衰减；向左拨动拨轮减小衰减。长按左/右进入快速连续步进模式。
  • 功能切换：短按拨轮中心。
  • 衰减调节：屏幕左上角显示<数值>，拨轮用于控制主衰减值。
  • 步进设置：左上角<数值>反显，拨轮用于调节步进大小（最小 0.25dB）。

5.3 射频开关控制操作（仅 SWITCH 模式）

主界面将显示 RF1/RF2 通道状态及 ENABLE 使能状态。选中的连通路径会以黑底白字反显。

  • 通道切换：向左拨动选择 RF1；向右拨动选择 RF2。
  • 中键功能：根据菜单中 Mid Key 的设置，短按拨轮中心可执行"开启/关闭使能"或"一键切换通道"。
  • 死锁唤醒：当设备处于 Disable（禁用）且画面为空心状态时，按任意键（左/中/右）将优先触发"开启使能"，设备瞬间恢复工作状态。

7. ZYE660 衰减模块

ZYE660 是基于 PE43711 的数字程控射频衰减模块，负责射频衰减通道。它可作为单模块使用，也可与 ZYC100 控制器组成积木式测试系统。

参数说明
频率范围9 kHz – 6 GHz
衰减范围0 至 31.75 dB
衰减步进0.25 dB
核心器件PE43711
射频接口SMA 输入 / 输出
版本形态单模块版；离线版（升级版，内置电池）

7.1 单模块版

单模块版适合二次集成、模块备件、方案验证和与 ZYC100 控制器组合使用。射频衰减通道由 ZYE660 提供，控制逻辑由 ZYC100 或外部系统完成。

7.2 离线版（升级版）

离线版（升级版）内置电池，适合现场便携测试和脱机使用，同时保留数字程控衰减能力。该版本可独立便携使用，也可接入 ZYC100 生态。

8. ZYA1000 上位机软件

ZYA1000 是 ZYC100 系列的桌面控制软件，基于 Rust + Tauri 构建，运行于 Windows 和 macOS。当前说明基于 ZYA1000 v1.4.1 的功能整理。

连接设备自动发现 USB CDC 设备，也可手动添加 UART 串口设备。
日常控制设置衰减、RF 通道、输出使能、设备别名和日志保留时间。
测试扩展支持多设备平铺、插入损耗补偿、TCP Bridge 和自动化时间线。

8.2 安装与启动

8.3 主界面布局

初次使用时，建议先关注左侧设备列表和右侧衰减控制卡片；确认通信正常后，再使用多设备平铺、补偿和自动化测试功能。

8.4 REL/TRU 与插入损耗补偿

REL 表示相对衰减，直接设置衰减器衰减值。TRU 表示真实衰减，上位机根据插入损耗补偿数据计算补偿后的控制值，使实际链路接近目标真实衰减。

  • CSV 查找表：两列数据（频率 GHz, 插入损耗 dB），按频点线性插值。
  • 公式：例如 S21 = -1.17217 - 0.30125f - 0.01049f²，其中 f 为频率（GHz）。
  • Touchstone S2P：自动读取 S21 并进行频率插值，支持 DB、MA 和 RI 格式。

9. 连接与日常操作

9.1 典型连接流程

  1. 确认 ZYC100 控制器、ZYE660 模块和射频链路连接正确。
  2. 将 ZYC100 通过 USB 连接电脑，或通过 USART1 接入外部控制系统。
  3. 打开 ZYA1000，上位机会自动扫描 USB CDC 设备。
  4. 选择设备后设置衰减值、步进值、RF 通道或输出状态。
  5. 需要多设备一致动作时，开启同步控制后再操作。

9.2 衰减设置规范

  • 范围：0.00 至 31.75 dB。
  • 步进：0.25 dB。
  • 输入超出范围时，系统会按有效范围和量化规则自动处理。
  • 自动化扫描建议先用较低速率验证链路和仪表响应，再提高速度。

9.3 本机操作 vs 上位机操作

场景推荐方式
现场快速调试本机 OLED + 拨轮操作（ATT/SWITCH 模式）
多设备批量控制ZYA1000 上位机（平铺视图 + 同步控制）
自动化重复测试ZYA1000 自动化测试工具（时间线 + 波形扫描）
第三方系统集成TCP 控制桥 127.0.0.1:19100 或 USART1 AT 指令
固件升级固件升级工具（Y-Modem 或 ST-Link）

10. 自动化测试与外部控制

10.1 TCP 控制桥

ZYA1000 启动后在本机开启 TCP 控制桥，默认地址为 127.0.0.1:19100。第三方程序（LabVIEW、Python、自动化框架等）可通过该端口向在线设备发送公开 AT 指令：

at+Connect?
at+GetAttenuationVal?
at+SetAttenuationVal=`10.25`
at+GetBattery?

多设备寻址：

zya:dev=SERIAL;at+GetBattery?
zya:send=usb:SERIAL;at+SetAttenuationVal=`10.25`
ZYA1000::SERIAL::INSTR at+Connect?
安全限制：控制桥默认拒绝内部维护类高风险 AT 指令，仅开放公开操作指令。桥接服务仅绑定 127.0.0.1，不可从外部网络访问。

10.2 JSONL 协议示例

{"cmd":"hello"}
{"cmd":"list"}
{"cmd":"send","at":"at+Connect?"}
{"cmd":"send","device":"SERIAL","at":"at+SetAttenuationVal=`10.25`"}

10.3 自动化时间线

菜单「工具 → 自动化测试工具…」打开独立窗口，提供以下能力：

  • 命令行：发送单条 AT 指令，可设置延迟、条件判断和设备选择器。
  • 循环：支持按次数、按持续时间或按回复条件结束，支持嵌套。
  • 变量：可在命令中使用 ${变量名} 或 {{变量名}} 进行动态替换。
  • 波形扫描：按绝对时间轴执行高速衰减扫描，支持 dryRun 估算频率和时序统计。
  • 配置管理：可保存、导入和复用测试配置，默认目录为 Documents/ZYA1000/configs。

11. 远程控制协议 (AT 指令集)

设备支持 Type-C (USB CDC) 和 UART (USART1) 接口同步控制，默认波特率为 115200 bps，所有指令必须以 \r\n（回车换行）作为结束符。

11.1 基础设备指令

指令响应说明
at+Connect? at+Connect=D1, hw3.31-B0001-D229AA08, sw2.3 确认设备连接状态，获取硬件/固件/UID 版本
at+GetBattery? at+GetBattery=85, 3.95V, Charging, 450.5mA 返回电量百分比、电池电压、充电状态及电流

11.2 衰减器控制指令（ATT 模式下有效）

操作类型指令响应说明
查询at+GetAttenuationVal?at+GetAttenuationVal=10.25获取当前衰减值
设置at+SetAttenuationVal=10.25ok / error: step设置目标衰减值；步进错误会返回 error
TRU 设置at+SetTrueAttenuationVal=10.25,11.42ok设置真实衰减目标值与补偿后控制值（TRU 模式）
提示：手动调节衰减拨轮时，设备会主动向串口推送实际衰减值指令。

13. 常见问题处理

13.1 上位机找不到设备

  • 确认 USB 线缆支持数据传输（非仅充电线）。
  • 确认设备已上电并进入正常工作状态。
  • Windows 下确认 STM32 Virtual COM Port 驱动正常。
  • 尝试重新插拔 USB，或在 ZYA1000 中点击刷新按钮手动扫描。

13.2 数值无变化（衰减器）

  • 检查衰减器 PS 拨码开关是否设置为 1 状态。
  • 确认 Mode 菜单已设置为 ATT 模式。
  • 检查通信日志中是否有错误回包。

13.3 开关无动作（射频开关）

  • 检查 4Pin 连接线是否插紧。
  • 确保菜单中的 Mode 已设置为 SWITCH。
  • 确认 VCC SET 开关已设为 VBUS 模式（5V 供电）。

13.4 通讯无回复

  • 在串口助手中勾选"发送新行"或"加回车换行"，确保指令携带 \r\n 结束符。
  • 确认波特率设置为 115200。

13.5 刚上电按键无反应

  • 系统刚启动时会进行设备类型侦测，请等待屏幕出现主界面后再进行操作。

13.6 电池信息与设备不一致

ZYC100 控制器本体不含电池。电池信息来自当前连接设备或模块的能力，例如 ZYE660 离线版（升级版）。如果某些设备不支持电池查询，上位机可能显示不可用或无数据。

13.7 衰减值无法设置

  • 确认设备在线。
  • 确认目标值在 0.00 至 31.75 dB 范围内。
  • 检查通信日志是否出现错误回复。
  • 如果使用 TRU 模式，确认补偿数据已导入且频率范围有效。

13.8 自动化测试不稳定

  • 先用快速测试确认连接方式和设备选择器正确。
  • 降低扫描频率，确认仪表和链路响应正常。
  • 使用 dryRun 检查理论频率和时间线配置。
  • 多设备并行时，建议使用 USB CDC 并保持端口稳定。
